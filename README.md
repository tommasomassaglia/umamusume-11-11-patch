# Uma Musume Skill Database

A comprehensive web-based database showcasing Uma Musume Pretty Derby skills with English translations and detailed information.

## Features

- **Complete Skill List**: Browse all Uma Musume skills with their effects and conditions
- **English Translations**: Skills translated from Japanese to English
- **Character Icons**: Visual representation of characters associated with each skill
- **Detailed Information**: View skill conditions, duration, cooldown, and effects
- **Responsive Design**: Works on desktop and mobile devices

## About

This is a static website that displays Uma Musume Pretty Derby skill information in an easy-to-read format. The data includes:

- Skill names and IDs
- Effect descriptions
- Activation conditions
- Duration and cooldown times
- Character associations
- Visual skill icons

## Structure

- `umamusume_eng.html` - Main HTML file with the skill database
- `LK2N6LOIOQXQFNLISWFUZA3PZ2S3JW2C_files/` - Assets directory containing:

  - Character icons
  - Skill icons

## Usage

Simply open `umamusume_eng.html` in your web browser to view the skill database locally, or visit the GitHub Pages link above for the online version.
